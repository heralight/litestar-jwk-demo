# Python Litestar and OIDC providers

This is POC to demonstrate the integration with OIDC providers like Zitadel, Keycloak, Oauth, ...etc.

## Deps

- PDM

```sh
pdm venv create
. .venv/bin/activate
pdm install
```

## Configuration

Set your settings inside .env file.

## Run

```sh
litestar run -d
``````
